BreakScope Ver9.0 2017/6/24

ブレイクスコープをダウンロードしていただきありがとうございます。

本作品は潜入要素の混じった、少し稀な雰囲気のシューティングゲームです。
ところが、ふつうのゲームとして楽しんでもらうほか、内部のステージや敵・武器・弾・ギミック・エフェクトまで
を自由に変更し、自分なりのゲームにしてしまう機能もついています。
ゲーム制作やJava言語の学習用としても役に立てたとしたら幸いです。

起動にはJava８ランタイムパッケージが必要ですので、ご了承ください。

パッケージダウンロードサイト：
http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
(英語サイトですが、日本語でのインストール手順がWebで検索できます)

<ゲーム>

ゲーム本体を起動するには
「ゲーム起動」ショートカットを開いてください。
他のショートカットでもありえることですが、開く際に警告がでる場合があります。気にせず続行してください。

操作方法：
射撃：クリック
連射：クリック長押し
移動：「W」「A」「S」「D」または矢印キー
リロード：「R」
パンチ＆レバー操作＆バイク乗降：「Space」
低速移動：「Shift」長押し中に移動
武器切り替え：マウスロール、数字キー、武器欄からクリックのどれでも可
ポーズ：「Esc」「P」
//////////////////////
デバッグ：「F3」
静止状態：「F5」

本作品にストーリー要素はありません。２５面あるステージの好きなところから始められます。
集めた弾薬はそのステージでしか使えません。

全体的に難しめで、一回でクリアするのはなかなか辛いと思います。
左上にクリア進行度、左下にHPと残機が表示されているので、それを目安に頑張ってください。

こつは、壁をどう壊すかにあります。
どのステージも敵が圧倒的に多いですが、同時に多くの壁が視線をふさいでくれています。
跳弾や奇襲をうまく繰り返し、敵の殲滅を目指してください。

よくわかんないものがあったら、とりあえずスペースキーを押してみてください。
敵だとふっとばせて、レバーだと操作することができ、バイクなどなら乗る/降りることができます。

<エディター>
エディターを使って任意のステージを編集し、壁や敵の位置や数を自由に変更できます。
「エディター起動」ショートカットから開けます。

操作方法：
保存：ポーズ画面から「セーブ」ボタンか「Ctrl + S」
選択：物体をクリック
移動：ドラッグか、選択状態で矢印キー
視点移動：選択状態を解除した上で、矢印キー
削除：「Delete」「BackSpace」
設置：数字キー
設置物スクロール：「E」(敵とギミックのみです)
設置物切り替え：左下の黒丸をクリック

他の詳しい操作方法はその都度表示されます
//////////////////////
全ステージ再保存：　ステージ選択画面で「F12」

ステージをまっさらにしたいときは、「ステージデータ」ショートカットから目的のファイルを削除してください。
一度にすべて消せば全ステージがまっさらになります。

<コンフィグ>
ステージの配置だけでなく、内部のユニットや攻撃もある程度変更できます。
特定フォルダ内の.iniファイルを編集して、敵・ギミック・武器・弾・エフェクトなどを自由に追加、改変することができます。
Javaの知識がない方でも、数字をいくつか入れるだけで、ブレイクスコープの世界をいじることができます。
既存のものを削除するとステージが読み込めなくなる危険があります。

「コンフィグ」ショートカットより
bullet  -  弾
effect  -  エフェクト
enemy   -  敵
gimmick -  ギミック
weapon  -  武器
の各フォルダーから、個別の「記入例.txt」がありますので、ご参考ください。
編集後、「コンフィグ更新」ショートカットを開いて、何事もなく閉じれたら反映完了です。

<画像データ・音声データ>
そのままファイル名を変えずに任意の画像・音声の内容を変えると、すぐにゲームやエディターに反映されます。
コンフィグで新しく敵や弾を追加するとき、指定する画像・音声はこのフォルダから探されます。
