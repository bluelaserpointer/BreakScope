__コンフィグ2.0__書き方説明 BS:1.9.0

コンフィグ2.0は、武器や弾、エフェクトを同一のiniファイルに書き込むことを可能とするバージョンです。
現在のBS1.9.2=beta9.2では、この形式のみmodファイルへ生成できるようになっています。

１．iniファイル内で指定される画像や音声ファイルは、そのフォルダ内で探されます。見つからなかったとき、config1.0でいつも通り使用していた場所を探します。
２．必要な画像や音声ファイルをすべて__config2.0__内に移動できれば、前バージョンより簡単にコンフィグを配布することができます。
３．それぞれのオブジェクトの上に[オブジェクト種類]を書くことで、異なる種別（武器や弾など）を同一ファイルに書き込むことができます。
４．(※実装中※)MODを導入している場合、そのMOD内にあるオブジェクトを"[MOD名] :: [オブジェクト名]"で使うことができます。
　起動時、そのMODが実装されていないと通常通りの警告が発生し、起動自体に影響を及ぼしません。

例：ロケット花火.ini　(内容：武器１個、弾２個、他MODから引用したエフェクト２箇所）

[weapon]
Name = ロケット花火
BulletKind = 赤弾,青弾

[bullet]
Name = 赤弾
DestroyEffect = 爆発系エフェクトMOD[BS1.9.0]　:: 赤い爆発
Burst = 80

[bullet]
Name = 青弾
DestroyEffect = 爆発系エフェクトMOD[BS1.9.0]　:: 青い爆発
Burst = 80
