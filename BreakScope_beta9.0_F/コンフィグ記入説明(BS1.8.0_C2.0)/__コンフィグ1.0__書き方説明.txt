__コンフィグ1.0__書き方説明 BS:1.9.0

コンフィグ1.0は、武器や弾、エフェクトなど、ゲーム内に登場する要素を簡単に変更できるシステムの初代バージョンです。
このバージョンは作りやすさを優先したシステムのため、配布には他のバージョンに比べると不向きです。
本格的な武器制作やMOD開発前の練習用として使えます。

１．iniファイル内で指定される画像や音声データは、それぞれ「picture」、「media」内で探されます。見つからなかったとき、警告が表示され、これらのデータ使用をスキップします。
２．武器・弾などのコンフィグ種別はフォルダごとで区分されています。
３．項目の順番は特に決まりがありません（一応、Nameなどの重要情報は先に書かれます）。

例：ロケット花火.ini　(内容：武器１個、弾２個、他MODから引用したエフェクト２箇所）

[ロケット花火]
Name = ロケット花火
BulletKind = 赤弾,青弾

[bullet]
Name = 赤弾
DestroyEffect = 爆発系エフェクトMOD[BS1.9.0]　:: 赤い爆発
Burst = 80

[bullet]
Name = 青弾
DestroyEffect = 爆発系エフェクトMOD[BS1.9.0]　:: 青い爆発
Burst = 80
