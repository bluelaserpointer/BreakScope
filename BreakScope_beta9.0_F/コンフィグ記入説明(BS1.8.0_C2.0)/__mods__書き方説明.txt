__mods__書き方説明 BS:1.9.0

modは、完全に独立したフォルダを作ることにより、他人への自作武器の配布がより簡単になるシステムです。
現在のBS1.9.2=beta9.2では、__config2.0__形式のみmodファイルへ書き込めるようになっています。

１．フォルダ内に"__config2.0__"というフォルダを作り、その中に通常のコンフィグを入れてください。
２．使用する画像や音声ファイルもそのフォルダ内に入れることができます。
３．また、使用する追加コードもそのフォルダ内に入れることができるので、自作敵・エンティティ・ギミックの配布もできます。
４．現在のバージョンでは対応していませんが、MODごとにMANIFEST.iniを作ることで、このMODの作者や配布日時などの情報をゲーム内で確認することができます。
５．ファイル構造の例は以下のようになります。
BreakScope_beta9.2
┗beta9.0
 ┗source
　　┗__mod__
　　　┗自作MODv1.0（これより下が自作部分）
　　　　┣__config2.0__
        ┃┗自作武器v1.0.ini
	┗MANIFEST.ini
６．modフォルダ内に存在しないファイルをコンフィグ内で指定すると警告が表示されて、通常通りconfig1.0時代の参照場所を見てくれます。
配布する前に、これらの警告は必ずすべて解決してください。

なお、現在のMANIFESTの書き方は一応こうなっています。

Name = 名前
Version = バージョン（自分のMODのバージョン、ブレスコではない）
Author = 作者
ReleaseDate = 配布日時
RequestBSVersion = 要求ブレスコバージョン(beta9.0とか1.9.0とか)
RequestMod = 要求MOD(あるMODがないと動作しない！などのときに使う)
BannerImg = バナーの画像
Description = このMODの説明

今後、これを表示するシステムを入れる予定です。

また、「HellFistsMod[BS1.9.0]」というブレスコbeta9.0のデフォルト武器MODを入れましたので、こちらもご参考ください。
