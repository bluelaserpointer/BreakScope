# BreakScope
よくある感じの２Dシューティングゲームです。A tipical 2D shooting game made by Java.

[中文]
 这是用Java制作的2D潜入类射击游戏。
 本作品没有使用任何游戏引擎。
 这个游戏的作者可以说日文也可以说中文，但对这个作品内的文章（日文）并没有进行任何翻译，请谅解。
 欢迎使用此程序做为制作Java游戏的参考。
